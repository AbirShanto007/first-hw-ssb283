Type of html tags -

There are two types of tags in html.

  1. Container tag
  2. Empty tag

Disscuss about the basic "html" tag -

  • !doctype html
    • It represents the document type, and helps browsers to display web pages correctly.
    • It must only appear once, at the top of the page (before any HTML tags).
    • The declaration is not case sensitive.
  • html
    • It represents the root of an HTML document.
    • tag is the container for all other HTML elements (except for the '! DOCTYPE' tag).
  • head
    • is a container for metadata (data about data) and is placed between the "html" tag and the "body" tag.
    • The following elements can go inside the "head" element:
      • title
      • style
      • base
      • link
      • meta
      • script
      • noscript
  • link
    • Defines the relationship between the current document and an external resource.
    • Is most often used to link to external style sheets.
  • title
    • This tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
    • Required in HTML documents!
    • Defines a title in the browser toolbar
  • body
    • Defines the document's body.
    • Contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
    • There can only be one "body" element in an HTML document.
  • img
    • Is used to embed an image in an HTML page.
    • has two required attributes:
      • src - Specifies the path to the image
      • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed
  • h1 - h6
    • Are used to define HTML headings.
    • h1 - defines the most important heading. h6 - defines the least important heading.
  • p
    • Defines a paragraph.
    • Browsers automatically add a single blank line before and after each "p" element.
  • br
    • Inserts a single line break.
    • Is useful for writing addresses or poems.
  • b
    • Specifies bold text without any extra importance.
    • According to the HTML5 specification, the "b" tag should be used as a LAST resort when no other tag is more appropriate.
  • a
    • Defines a hyperlink, which is used to link from one page to another.
    • The most important attribute of the "a" element is the href attribute, which indicates the link's destination.
  • ol
    • Defines an ordered list. An ordered list can be numerical or alphabetical.
  • ul
    • Defines an unordered (bulleted) list.
  • li
    • Defines a list item.
    • The "li" tag is used inside ordered lists(ol), unordered lists (ul), and in menu lists (menu).
